Sobre o Sustentabilidade Tapera

O Sustentabilidade Tapera nasceu para unir tecnologia e cidadania, focando na melhoria da infraestrutura e saneamento do nosso bairro.

Nossa Missão

Proporcionar uma ferramenta ágil onde o cidadão possa reportar problemas ambientais, garantindo que as autoridades locais tenham dados precisos para uma atuação eficiente.

Tecnologias Utilizadas

  • Backend: Java Spring Boot
  • Banco de Dados: PostgreSQL
  • Frontend: HTML5, CSS3, Bootstrap 5
  • Mapas: Leaflet.js / OpenStreetMap
  • Segurança: JWT (JSON Web Tokens)
